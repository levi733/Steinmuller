# Mondi CFB Boiler Dashboard

Static HTML dashboard for the Mondi CFB Boiler Jan-Apr 2026 productivity analysis.

## Run locally

From this folder:

```powershell
python -m http.server 4173
```

Open:

```text
http://127.0.0.1:4173/
```

## Deploy

This folder is ready to push to Git and deploy as a static site. The site entry point is `index.html` at the repository root.

No build step, npm install, database, or environment variables are required.
